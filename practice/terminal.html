Terminal tutorial

Vocabulary

Command line
A special place in your computer where you can alter the computer and how it functions.
Command
A command is a special set of instructions that the computer runs once called.
Argument
An argument is a part of a command that gives the command special instructions to do something besides its default.
Flag
A special type of argument that alters how the command is run. Flags are signaled with a dash.

Terminal commands

  • ls
    • Lists the contents of the directory. If no directory is specified, lists the contents of the current working directory.
    • How to use: ls $DIR. $DIR is the directory. Use the -l flag to get more information like ls $DIR -l.
  • cd
    • Changes the current working directory to the directory you choose. In effect, moves you around the computer.
    • How to use: cd $DIR. $DIR is the directory you want to change to. Use the .. argument to go back like cd ...Use cd to go to your home directory.
  • pwd
    • This command shows were you are in the computer all the way up to the top
    • How to use: pwd.
  • mkdir
    • Makes a new directory in your current working directory
    • How to use: mkdir $DIR. $DIR is the name of the directory.
  • rm and rmdir
    • Removes a file or directory. Be careful when using this command because there is no way to get the file back.
    • How to use: rm $FILE, rmdir $DIR. $FILE is the file you want to delete, and $DIR is the directory you want to delete.
  • cp
    • Copies a file and moves it to a different directory
    • How to use: cp $FILE $LOCATION. $FILE is the file you want to copy, and $LOCATION is where you want to move the file.
  • mv
    • Moves a file to a different directory.
    • How to use: mv $FILE $LOCATION. $FILE is the file you want to move, and $LOCATION is where you want to move the file.
  • grep
    • Searches for specific text in a file
    • How to use: grep "txt" $FILE. Txt is the specific text that you want to search for in a file, and $FILE is the file you want to search for that text.
  • less
    • Displays the contents of a file
    • How to use less $FILE. $FILE is the file you want to see. Use the up and down arrows to look through the file and use q to quit.
  • man
    • Gives you all of the information on how to use a command.
    • How to use: man $UTIL. $UTIL is the command you want to get information about. Scroll through the manual with the up and down arrows and use q to quit.
  • touch
    • A command that creates a file
    • How to use: touch $FILE. $FILE is the name of the file that you want.
  • exit
    • A commands that exits the terminal
    • How to use: exit
  • open
    • Opens a file or a directory on Finder or a different application
    • How to use: open $FILE, open $DIR. $FILE is the file you want to open, and this file will open in the default app that opens their document type. $DIR is the directory you want to open, and it will open in the Finder app.
  • sudo
    • A very powerful command that lets you run commands with super user privileges.
    • How to use: sudo $CMD. $CMD is the command you want to run with its arguments. When using sudo, the computer will ask you for a password, which is so secure that it doesn't show any characters at all, just a key icon. Sudo is prohibited by APS.
