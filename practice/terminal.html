Terminal tutorial

Vocabulary

Command line
A powerful tool that lets you change basically anything in your computer
Command
A special phrase that when used, tells the computer exactly what to do
Argument
A phrase added to a command that alters how the command is run

Terminal commands

  • ls
    Lists the contents of the directory. 
    ls $DIR

    $DIR is the directory. Use the -l flag to get more information like:

    ls $DIR -l
  • cd
    Changes the current working directory.
    1. Type cd followed by the directory name.
    2. Use cd .. to go back one level.
    3. Use cd alone to return to your home directory.
    cd $DIR
  • pwd
    Shows where you are in the computer. 
    pwd

    Example output: /Users/student/projects

  • mkdir
    Makes a new directory. 
    mkdir $DIR
  • rm / rmdir
    Removes a file or directory. 
    rm $FILE
    rmdir $DIR

    $FILE is the file you want to delete, and $DIR is the directory you want to delete.

  • grep
    Searches for text in a file. 
    grep "txt" $FILE

    Example output: Found "txt" on line 42

  • less
    Displays the contents of a file. 
    less $FILE

    Navigate with ↑ and ↓, quit with q.

  • man
    Shows the manual for a command. 
    man $UTIL

    Scroll with ↑/↓, quit with q.

  • touch
    Creates a file. 
    touch $FILE
  • exit
    Exits the terminal. 
    exit
  • open
    Opens a file or directory. 
    open $FILE
    open $DIR
  • sudo
    Runs a command with superuser privileges. 
    sudo $CMD

    When prompted, type your password (it will not display as you type).

    The sudo command for your account is prohibited by APS.
